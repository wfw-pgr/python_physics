減衰補正 ( 連鎖崩壊 )
##############################################################

=========================================================
連鎖崩壊の式
=========================================================

---------------------------------------------------------
放射能の式
---------------------------------------------------------

.. math::

   N_A &= N_{A0} e^{ - \lambda_1 t } \\
   N_B &= \dfrac{ \lambda_2 }{ \lambda_2 - \lambda_1 } N_{A0} \left[ e^{ - \lambda_1 t } - e^{ - \lambda_2 t } \right]


---------------------------------------------------------
収量最大の時刻
---------------------------------------------------------

.. math::

   T_{max} &= \dfrac{1}{\lambda_1-\lambda_2} ln \left[ \dfrac{\lambda_1}{\lambda_2} \right]


   
=========================================================
補正式
=========================================================

時刻 :math:`t=t_1, t_2` における放射能の比を考えて補正する．

.. math::

   \dfrac{ N_B(t=t_1) }{ N_B(t=t_2) } &= \dfrac{  \dfrac{ \lambda_2 }{ \lambda_2 - \lambda_1 } N_{A0} \left[ e^{ - \lambda_1 t_1 } - e^{ - \lambda_2 t_1 } \right]  }{ \dfrac{ \lambda_2 }{ \lambda_2 - \lambda_1 } N_{A0} \left[ e^{ - \lambda_1 t_2 } - e^{ - \lambda_2 t_2 } \right]  } \\
   &= \dfrac{ e^{ - \lambda_1 t_1 } - e^{ - \lambda_2 t_1 } }{ e^{ - \lambda_1 t_2 } - e^{ - \lambda_2 t_2 } }

ここで、 :math:`t_1, t_2` をそれぞれ、収量最大の時刻 :math:`T_{max}` 、計測開始 :math:`T_{start}` と計測終了 :math:`T_{end}` の中間時刻として設定すれば、

.. math::

   t_1 &= \dfrac{1}{\lambda_1-\lambda_2} ln \left[ \dfrac{\lambda_1}{\lambda_2} \right] \\
   t_2 &= \dfrac{T_{start} + T_{end} }{2}

時刻を補正できる．
